# Home Assistant Add-on: Headscale

Repository di add-on per Home Assistant contenente
[Headscale](https://headscale.net/), implementazione open source e
self-hosted del control server di Tailscale, con la dashboard
[Headplane](https://github.com/tale/headplane) integrata (opzionale).

## Installazione

[![Aggiungi repository a Home Assistant](https://my.home-assistant.io/badges/supervisor_add_addon_repository.svg)](https://my.home-assistant.io/redirect/supervisor_add_addon_repository/?repository_url=https%3A%2F%2Fgithub.com%2FAlessioBazzanella%2Fhomeassistant-headscale-addon)

Oppure manualmente: **Impostazioni → Componenti aggiuntivi → Store → ⋮ →
Repository** e aggiungi:

```
https://github.com/AlessioBazzanella/homeassistant-headscale-addon
```

Poi installa **Headscale** dallo store. Le immagini sono precompilate
(`aarch64`/`amd64`) e pubblicate su GitHub Container Registry.
